## Blackjack game

Blackjack game created with the following packages:
- NextJs
- React
- Typescript
- TailwindCSS

## Getting Started

#### Clone repo

    git clone https://github.com/c0balt60/blackjack .

#### Install dependencies

    npm i

#### Run locally

    npm run dev

#### Run on website

    https://blackjack-demo-andrii.web.app/
